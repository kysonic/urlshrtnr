# URL SHRTNR

Hey! It is just an example of koa app.
 
## Prerequisites
 
1. node >= 6.9.0
2. mongodb >= 3.2.10
 
## Getting started 

1. Clone this repo to your computer: 

    `$ git clone https://github.com/kysonic/urlshrtnr.git .`
    
2. Run:
     
    `$ npm run prepare`
     
3. Run to launch app:

    `$ npm start`
